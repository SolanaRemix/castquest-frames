# CASTQUEST Constellation Overview

This document describes the purpose of each core repository in the CASTQUEST constellation.

- castquest-frames: Farcaster Frames and interaction surfaces.
- castquest-templates: Base templates for repos, docs, and flows.
- castquest-sdk: SDK for interacting with the protocol and contracts.
- castquest-games: Game logic and state machines.
- castquest-quests: Quest logic and progress tracking.
- castquest-contracts: Smart contracts for the protocol.
- castquest-ai: AI modules and automation.
- castquest-ui: Frontends, dashboards, and FX overlays.
- castquest-indexer: Event ingestion and indexing.
